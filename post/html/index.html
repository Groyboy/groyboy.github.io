初识html语言

从去年6月份开始我刚刚接触到前端，那个时候我就在倒腾微信小程序，那个时候甚至什么是html都不懂，也没有人带着多多少少走了很多弯路，最终也是什么都没有做出来知道后来，我看了很多前辈所写的前端学习笔记，那个时候我才知道前端学习的基础语言html、css、js等。所以自己也想写一些基础的语言的笔记正好复习一下。


什么是html


从官方的翻译来说html是（HyperTextMarkupLanguage——超文本标记语言的缩写）是构成Web世界的一砖一瓦，通缩易懂的说，就是浏览器语言。
举个例子来说：中国人有中文是自己的语言；英国人有英语是自己语言；那么浏览器也有自己的语言也就是html语言。


语言的基本结构


有过其他语言学习经验的朋友就会懂，什么是麻雀虽小却五脏俱全，也就是头-身体-尾巴

<!DOCTYPE html>   
    <html>          
    <head>        
        <title>这是一个新的标题</title>
    </head>          
    <body>        
   这是一个html代码
    </body>
    </html>

所有的HTML代码都放在一个<html>元素，<html>元素可以包含两个部分:<head>和<body>两个元素。
<head>元素用于包含整个文档的一般信息，比如文档的标题（<title>元素用于包含标题），对整个文档的描述，文档的关键字等等。
文档的具体内容就要放在<body>元素里。HTML使用标签对的方法编写文件，<标签名></标签名>（比如<head></head>标签对）


html 都包括哪些标签


通过不同的标签，HTML文档可以包含不同的内容，比如文本，链接，图片，列表，表格，表单，框架等。

  • 文本 ：HTML对文本的支持是最丰富的，可以设置不同级别的标题，分段和换行，可以指定文本的语义和外观，等等等等。
  • 链接 ：链接用来跳转当前页面到另一个页面，就好比传送门。
  • 图片 ：图片填充页面使其更加美观与充足，或提供更多的信息。
  • 列表 ：列表用于说明一系列条目是彼此相关的。
  • 表格 ：表格是按行与列将数据组织在一起的形式。也有人使用表格进行页面布局。
  • 表单 ：表单通常由文本输入框，按钮，多选框，单选框，下拉列表等组成，使HTML页面更有交互性。
  • 框架 ：框架使页面里能包含其它的页面。


浅谈html 常用元素


1、头部元素


标签描述
<hepd>定义关于文档的信息。
<title>定义文档标题。
<bpse>定义页面上所有链接的默认地址或默认目标。
<link>定义文档与外部资源之间的关系。
<metp>定义关于 HTML 文档的元数据。
<script>定义客户端脚本。
<style>定义文档的样式信息。

2、文字与段落


网页中经常需要显示文字，需要设置文字格式，如字号、字体、颜色、段落格式等

标签 描述
<html> 定义 HTML 文档。
<body> 定义文档的主体。
<h1> to <h6>定义 HTML 标题大小
<hr>定义水平线。
<!–>定义注释。
<p>定义段落。
<br />插入单个折行（换行）。
在html5中属性废除了html4的各种颜色字体的标签和属性，应该使用样式来代替，在写代码时要多注意查看文档


3、链接标签

1、超链接
<a href="" title="" target="">填写内容</a>
 （自身页面不关闭，打开一个新的链接页面）
href 去往的路径（跳转的页面） 必写属性
title 提示文本 鼠标放到链接上显示的文字
target=”self” （默认值） 在自身页面打开（关闭自身页面，打开链接页面） Target=”blank” 打开新页面
2、锚链接
先定义一个锚点：超链接跳转到页面的任何位置。一般用于在页面下面的时候，点击回到最上面。名称AAA可以随意区，只要是标记作用。
    <p id="AAA">

</p>
...
<a href="#AAA"></a>   // 超链接到锚点
3、空链接
不知道链接到那个页面的时候，用空链
<a href="#">空链</a>
4、超链接优化
<base target="_blank">写在head里头每个链接新开一个窗口


音乐、图片插入


标签描述
<img>定义图像。
<map>定义图像地图。
<area>定义图像地图中的可点击区域。
<audio>标签定义声音，比如音乐或其他音频流。
<embed>标签定义嵌入的内容，比如插件。

特殊字号


标签显示描述
&lt;<小于号或显示标记
&gt;>大于号或显示标记
&amp;&可用于显示其它特殊字符
&quot;“引号
&reg;®已注册
&copy;©版权
&trade;™商标
&ensp;半个空白位
&emsp; 一个空白位
&nbsp; 不断行的空白
