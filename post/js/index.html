js介绍

js为什么会诞生？由于网景公司希望能在静态HTML页面上添加一些动态效果，于是叫BrendanEich这哥们在两周之内设计出了JavaScript语言。你没看错，这哥们只用了10天时间。JavaScript除了语法上有点像Java，其他部分基本上没啥关系。

ECMAscript？


因为网景开发了JavaScript，一年后微软又模仿JavaScript开发了JScript，为了让JavaScript成为全球标准，几个公司联合ECMA（European Computer Manufacturers Association）组织定制了JavaScript语言的标准，被称为ECMAScript标准。


实现简单的js代码


<!DOCTYPE html>
    <html>
    <head>
        <title>javascript 代码实现</title>
        <script type="text/javascript" src="path/index.js"></script>
    </head>
    <body>
        <script type="text/javascript">
            alert('Hello, world');
        </script>
    </body>
    </html>


这段代码你可以很清楚的看到head部分中<script type="text/javascript" src="path/index.js"></script>这段代码就是引入外部资源，你可以在src=“”这里可以引入外部资源,可以指定绝对url、相对url，绝对url是指向其它站点（比如src=”https://cdn.jsdelivr.net/npm/vue"），相对url是指向站点内的文件（比如src="path/index.js"）。
在body部分中有<script type="text/javascript">alert('Hello, world');</script>这样一段代码，javascript可以嵌入在网页内部的任何地方，他被浏览器直接执行，不过我们通常把js代码都放在head当中。你在script代码中可以看到alert('Hello, world');这条代码可以在浏览器中实现输出Hello，world对于初学者来说可以自己亲自尝试一下。

基础语法


JavaScript语法是一套规则，它定义了JavaScript的语言结构，JavaScript语句定义了两种类型值：混合值和变量值，混合值被称为字面量，变量值被称为变量。

字面量


JavaScript中书写字面量重要的规则是，书写数值是否有小数点均可比如12.90,100，字符串是文本：由””or’’包围比如：”hello”,”world”


JavaScript变量


在编程语言中变量用于存储数据值。JavaScript用var来声明新的变量值“=”用于为变量赋值比如：var x = 7;,var y;所有JavaScript变量必须以唯一的名称标识，这些唯一的标识，称之为标识符，名称可包含字母、数字、下划线和美元符号，名称可以以字母开头、$、_开头，名称对大小写敏感，保留字（比如 JavaScript 的关键词）无法用作变量名称。“=”等号赋值是将右边的数据赋值给左边，如果原有的变量有赋值将会被替代掉


JavaScript运算符


JavaScript算数运算符，算数运算符用于对数字执行算数运算：+（相加）、-（相减）、（相乘）、/（相除）、%（取余）、++（递加）、–（递减），我想你只要上过小学你就会明白四则运算是干什么的了，在这里讲解一下取余运算还有递加递减

<script>
    var x = 10;
    var y = 9;
      document.write(x%y)//    取余运算相当于用10/9=1.1....取1所以返回值为1
      var x = x++;
      document.write(x) // 返回值为10 x = x+1 因为是先返回在运算所以结果是10
      var x = ++x;
      document.write(x) // 返回值为11  ++x与x++相同 但是是先x+1在返回所以结果是11
      递减与递加原理相同 只是x = x-1
</script>
